# UniEats - University Canteen Ordering System 🍱

### Module: PUSL2023 - Mobile Application Development 📱 

---

## 📌 Project Overview
This project is a **robust, multi-tiered mobile application** developed using **Flutter**. It is designed to streamline the food ordering and payment process at university canteens, addressing real-world dining challenges by eliminating long queues and providing a secure digital transaction platform for students and staff.

---

## ✨ Key Features
* **Secure Authentication 🔐**: Integrated user login and registration system.
* **Digital Menu 📜**: Real-time browsing of available food items and pricing.
* **Smart Cart 🛒**: Effortless selection and management of food items for checkout.
* **Seamless Payments 💳**: Integrated secure payment options for a completely cashless experience.
* **Order History 🕒**: Tracking and viewing previous meal transactions and receipts.

---

## 🛠 Tech Stack
* **Frontend Framework**: Flutter (Dart) 🚀 
* **UI/UX Design**: Figma 🎨 
* **Database**: Firebase / Local Database Integration 💾 
* **Version Control**: Git & GitHub (Individual & Group Commits) ⚙️

---

## 🎨 UI/UX Design
The application interface is designed following **industry best practices** for mobile human-user interfaces. The layout focuses on accessibility and efficiency for university students and canteen staff.
