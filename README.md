UniEats - University Canteen Ordering System 🍱Module: PUSL2023 - Mobile Application Development 📌 Project OverviewThis project is a robust, multi-tiered mobile application developed using Flutter  to streamline the food ordering and payment process at university canteens. It addresses real-world dining challenges by eliminating long queues and providing a secure digital transaction platform for students and staff.✨ Key FeaturesSecure Authentication 🔐: User login and registration system.Digital Menu 📜: Real-time browsing of food items and pricing.Smart Cart 🛒: Effortless selection and management of food items.Seamless Payments 💳: Integrated secure payment options for a cashless experience.Order History 🕒: Tracking and viewing previous meal transactions.🛠 Tech StackFrontend Framework: Flutter (Dart) UI/UX Design: Figma Database: Firebase / Local Database Integration Version Control: Git & GitHub (Individual & Group Commits) 🎨 UI/UX DesignThe application interface is designed following best practices for mobile human-user interfaces.
