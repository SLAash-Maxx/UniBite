## 📌 Project Overview
[cite_start]This project is a **robust, multi-tiered mobile application** developed using **Flutter**[cite: 9, 43]. [cite_start]It is designed to streamline the food ordering and payment process at university canteens, addressing real-world dining challenges by eliminating long queues and providing a secure digital transaction platform for students and staff[cite: 14, 16].

---

## ✨ Key Features
* [cite_start]**Secure Authentication 🔐**: Integrated user login and registration system[cite: 13, 14].
* [cite_start]**Digital Menu 📜**: Real-time browsing of available food items and pricing[cite: 30].
* [cite_start]**Smart Cart 🛒**: Effortless selection and management of food items for checkout[cite: 30].
* [cite_start]**Seamless Payments 💳**: Integrated secure payment options for a completely cashless experience[cite: 14].
* [cite_start]**Order History 🕒**: Tracking and viewing previous meal transactions and receipts[cite: 30].
